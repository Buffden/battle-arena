# Profile Service

Spring Boot service for managing user profiles, scores, and statistics.

## Port
8082

## Tech Stack
- Spring Boot 3.x
- Java 17
- MongoDB
- Redis (caching)

## What It Does
- User profile management
- Global score tracking (infinite, no level cap)
- Rank tier calculation (Valorant-style)
- Win/loss statistics
- Profile updates
