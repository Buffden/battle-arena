# Auth Service

Spring Boot service for user authentication and authorization.

## Port
8081

## Tech Stack
- Spring Boot 3.x
- Java 17
- MongoDB
- JWT tokens
- BCrypt password hashing

## What It Does
- User registration
- User login
- JWT token generation and validation
- Google OAuth login
- Password hashing
